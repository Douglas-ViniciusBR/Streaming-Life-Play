# 🎬 Life Play

Página de seleção de perfis para uma plataforma de streaming, desenvolvida como parte da Imersão Front-end da Alura.

## 📋 O Projeto

Uma página responsiva inspirada em Netflix e Prime Video, onde o usuário escolhe seu perfil para começar a assistir. A interface apresenta 4 perfis (Gustavo, Agatha, Vinícius e Victor) com imagens e nomes.

## 🛠️ Tecnologias

- HTML5
- CSS3 (Grid, Flexbox, Animações)

## 🎨 Design

- **Fundo**: #10141A (preto escuro)
- **Gradiente**: Laranja (#FFA100) → Laranja-Vermelho (#FF7700) → Magenta (#FF3F7F)
- Design responsivo para desktop, tablet e mobile
- Animações suaves ao hover nos perfis

## ✨ Funcionalidades Implementadas

- Grid responsivo com cards de perfil
- Efeito zoom ao passar o mouse
- Gradiente no título com animação de fade-in
- Legendas com efeito de cor ao hover
- Layout totalmente responsivo
- Sombras dinâmicas nas imagens

## 📁 Estrutura

```
├── index.html          # Página principal
├── style.css           # Estilos da aplicação
├── logo/               # Logo da plataforma
└── img-perfil/         # Avatares dos perfis
```
